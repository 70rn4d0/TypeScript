A command-line todo / task tracker written in TypeScript, run with Node.js.

- Display tasks in a table format.
- Accept user input and store it in a JSON file.
- Show the user their progress.
- Allow the user to check off items from the todo list.

the data input accept two part, title and description.



The application accepts user input via the command line, stores tasks in a JSON file, and allows users to track and update their progress.
